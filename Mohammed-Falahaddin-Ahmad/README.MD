Readify is a comprehensive eCommerce platform designed for book lovers. It features a robust administration system for managing inventory (books, authors, categories), specialized user roles, and a dynamic frontend for customers to browse, review, and purchase books with real-time updates via Livewire.

Features
Admin Dashboard: Real-time statistics, inventory management, and discount control.
Dynamic Book Catalog: Search, filter by price/category/rating, and advanced sorting.
Interactive Reviews: Verified purchase review system with dynamic star ratings.
Livewire Integration: Instant "Add to Cart" and "Add to Wishlist" without page reloads.
User Authentication: Secure registration and login flow with personalized profiles.
Responsive Design: Premium UI with smooth animations and layout optimizations.
Technologies & Tools Used
Backend: Laravel (PHP), MySQL
Frontend: Livewire, JavaScript (Vanilla, jQuery), Bootstrap, Blade Templates
Design/Theming: Custom CSS, Swiper.js, SweetAlert2
Tools: PHPMyAdmin, GitHub, Visual Studio Code
